Detailed Description

A Block object is the entire foundation of the most fundamental data structure in bitcoin, the Blockchain. A Block object is simply used to group and store Transaction objects and store them on the Blockchain.

All Blocks have three fields: the blocks, the hash, and the size. The blocks field is just a list of the serialized Transaction objects. The hash is the hash of all the Transaction's hashes appended together. The size is just the number of Transactions in the Block.
