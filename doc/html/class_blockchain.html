Detailed Description

A Blockchain object is the main data structure in OneCoin. A Blockchain object is simply a linked list used to store all the Block data types. Each Block is held together by a nonce. The nonce is just a way of ensuring that the Blocks won't be altered once they are added to the Blockchain.

Each Blockchain has only 3 fields: size, blocks, and json_data. The size is just the number of Blocks currently being held in the Blockchain. blocks is just the list of the Block objects. The Block objects are represented as a serialized JSON string. The json_data field is just the serialized string of the Blockchain instance, represented in JSON format.
