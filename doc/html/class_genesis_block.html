Detailed Description

A GenesisBlock is a subclass of the Block class. A GenesisBlock holds all the same data and methods as a Block type. The main difference between A GenesisBlock and a Block is that a GenesisBlock can only be first in a Blockchain. In fact, the first Block in every Blockchain must be a GenesisBlock. Everything in a GenesisBlock is the same as a normal Block except that the last_block, the hash of the last block in the Blockchain, is always zero.
